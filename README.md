## 📋 System Requirements

To run MemBrain smoothly, ensure your system meets these minimum requirements:

- **Operating System**: Windows 10 or later, macOS 10.15 or later, or a modern Linux distribution (Ubuntu, Fedora).
- **RAM**: At least 4 GB.
- **Storage**: Minimum of 500 MB available disk space.
- **Processor**: 2 GHz dual-core or better.

## 🛠 Features

- **Agentic Memory Architecture**: MemBrain employs hybrid retrieval architectures that blend various indexing strategies. These include lexical matching, dense vector embeddings, and knowledge graph traversal. This setup ensures robust memory retrieval for diverse inputs.

- **Context Awareness**: Enjoy context-aware interactions as MemBrain learns and evolves from user experiences over time.

- **User-Friendly Dashboard**: Manage your AI’s memory easily with intuitive controls.
